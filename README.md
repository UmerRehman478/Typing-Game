# Typing Speed Test Game ⌨️

Welcome to the **Typing Speed Test Game** Test your typing speed and accuracy.

---

## 🚀 Features
- Random paragraphs to type 🖊
- Tracks:
  - ⏱️ Time left
  - ❌ Mistakes
  - 📝 Words per minute (WPM)
  - ✍️ Characters per minute (CPM)
- Easy to reset and try again! 🔄
